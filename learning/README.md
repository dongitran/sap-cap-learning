# 📚 SAP CAP — Learning Notes

Tài liệu học, ghi chú, cheatsheet và bài tập thực hành.

```
learning/
├── roadmap.md          # Lộ trình tổng thể (đọc đầu tiên)
├── notes/              # Ghi chú khi học từng chủ đề
│   ├── cds-basics.md
│   ├── odata-guide.md
│   ├── security-xsuaa.md
│   └── ...
├── cheatsheets/        # Quick reference, tra cứu nhanh
│   ├── cds-syntax.md   # CDS language cheatsheet
│   ├── odata-queries.md
│   └── cds-cli.md      # Các lệnh cds hay dùng
└── exercises/          # Bài tập nhỏ, thử nghiệm CDS/OData
    ├── ex01-entities/
    ├── ex02-associations/
    └── ...
```

## Gợi ý cách dùng

- **notes/**: Viết ghi chú bằng tiếng Việt khi học xong một chủ đề — tóm tắt những gì mình hiểu
- **cheatsheets/**: Copy syntax hay dùng vào đây để tra nhanh, không cần vào docs
- **exercises/**: Thử nghiệm CDS nhỏ lẻ, chạy `cds watch` để xem kết quả ngay
